# Active Threads

---

## Thread Format

Each thread is an H3 under its section. Fields:

- **Type:** `project-blocked` | `stale-card` | `action-item` | `invoice-deadline`
- **Created:** YYYY-MM-DD
- **Last Surfaced:** YYYY-MM-DD (updated by morning job)
- **Source:** wikilink to the originating note/card
- **Target:** person or date this is blocked on (if applicable)
- **Summary:** one-line description
