# Brand Voice

> [!info] Writing rules for cowork-generated content
> Every write-callout and write-summary sub-skill reads this file before composing output. Edit to taste; the rules below are the platform default for the life scope.

## Tone

Direct. Concise. No fluff. This is a personal briefing, not a newsletter.

## Rules

- No emojis. Ever. Not even one.
- No em dashes. Use hyphens or commas.
- No corporate speak ("leverage", "optimize", "actionable insights").
- No cheerful filler ("Great news!", "Hope you're having a great day!").
- Don't over-explain. Say it once.
- Short paragraphs. Short sentences.
- If there's nothing to report in a section, say "Nothing notable" and move on.

## Report Writing Style

- Lead with what needs attention.
- Facts first, then context if needed.
- Use tables for structured data (transactions, calendar events).
- Use callouts for things that need action.
- Bold the important parts, not everything.

## Examples

Good: "3 unread messages from {{example_inner_circle_name}}. Last one was 2 hours ago asking about dinner plans."
Bad: "You have some exciting messages waiting for you! {{example_inner_circle_name}} sent you 3 messages -- check them out!"

Good: "Spent $47.82 yesterday. $12.50 at {{example_merchant}} -- flagged as discretionary."
Bad: "Here's a comprehensive overview of your financial activity for the previous day..."

## Section Headers

Use inline Lucide SVG icons consistent with the vault style guide. No emojis in headers.
