## BeaconButton API contract (quoted, v0.6.0 lesson)

> [!info] From `platform/mechanisms/beacon-button/beacon-button.js:39`
> ```
> customJS.BeaconButton.render(parent, { label, icon, onClick, flex?, disabled?, tone? }) → HTMLButtonElement
> ```
> - `label` — string (required)
> - `icon` — string (required; inline SVG HTML)
> - `onClick` — function (required)
> - `flex` — boolean (optional; default false; set true to append `flex: 1; min-width: 0`)
> - `disabled` — boolean (optional)
> - `tone` — `"accent"` (optional; only valid value at v0.18.0; reserved slot)
