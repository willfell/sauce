# v0.13.0 Journal Blueprint Implementation Plan

> **For the implementing agent:** REQUIRED SUB-SKILL: Use `de:executing-plans` (parallel session) or `de:subagent-driven-development` (same session) to implement this plan task-by-task.

**Goal:** Ship `journal@0.1.0` blueprint mirroring headspace's Daily Journal pattern under `beacon/journal/YYYY/MM-MMMM/Journal-YYYY-MM-DD.md`. Nav-button-driven creation via existing `runTemplaterTemplate` action; manual creation auto-applies via `templater_folder_templates`. Adds one `journal` ICON entry to nav-buttons mechanism (2.3.0 → 2.4.0 MINOR).

**Architecture:** Single stage S1. No new mechanism API; no new action_type; no new helper. All foundations (split-field `runTemplaterTemplate`, `templater_folder_templates`, `applyTemplaterFolderTemplates`) shipped in v0.4.0/v0.4.2. Cycle is mechanically additive: one ICONS entry, one new blueprint manifest, one new template, three lockstep version bumps (mechanism manifest + workshop catalogue + workshop self-subscription + barebones subscription), workshop bump 0.12.0 → 0.13.0 MINOR.

**Tech Stack:** Node 18+ (install harness via `node platform/test/run-install.js <vault>`). Obsidian + Templater + nav-buttons mechanism (runtime). `platform/mechanisms/nav-buttons/space-nav-buttons.js` ICONS map insert at line ~52. Manual smokes in `tmp-test-barebones-vault` after Node-harness install (no Cmd+R required — no CustomJS class touched).

**Cycle exit criteria:**
- Workshop manifest at `0.13.0`; nav-buttons@2.4.0; journal@0.1.0; tag `v0.13.0` annotated + pushed.
- `run-install.js .` exit 0 + idempotent; `run-helper-cases.js` 255 still green; renderer harness regression-clean (use individual selectors per pre-existing T4.0 fatal).
- All S1 manual smokes A-E user-confirmed.
- Result writeup + CLAUDE.md status snapshot + post-v0.13.0 handoff committed; end-of-session protocol completed per `Docs/prompts/_handoff-protocol.md`.

**Constraints carried forward (read before any task):**
- Workshop_version bumps require user approval at commit time (CLAUDE.md ask-before-acting).
- Stub body untouchable (landmine #13); md5 invariant `a39257da1dd49ae4481e5cd0a42bdac4`.
- Landmine #12 allowlist stays at 3 paths; helper count stays at 4.
- Mobile-aware visual design (gotcha 7) — new icon must render correctly on narrow viewport.
