# v0.36.0 + v0.36.1 — execution log

> [!abstract] One-paragraph summary
> Sauce now distributes via Homebrew. `pantry/` lives in `$(brew --prefix)/Cellar/sauce/<v>/libexec` instead of inside synced vaults. A per-machine `~/.sauce/vaults.json` registry tracks installed vaults; `~/.sauce/active-pantry` symlinks an optional dev checkout. Six new CLI verbs (`vault`, `reinstall`, `migrate-layout`, `doctor`, `link`, `unlink`) operate the new surface. Tap repo `willfell/homebrew-sauce` ships `Formula/sauce.rb`; GitHub Actions auto-bumps the formula on `v*.*.*` tag pushes. Fixes the dirty-tree + sync-corruption second-machine failures from the pre-v0.36 layout.
