# v0.16.0 Finance Blueprint — Implementation Plan

> **For the implementing agent:** REQUIRED SUB-SKILL: `de:executing-plans` (parallel session) OR `de:subagent-driven-development` (same session). Per-task subagent dispatch on the 9 helpers + 10 templates; master-driven on mechanism patches + manifest + version bumps. Stage-close `AskUserQuestion` checkpoints between S1↔S2 and S2↔S3.

**Goal:** Ship `finance@0.1.0` blueprint covering personal (Budgets + Paychecks) AND business (Invoices) under one module at `beacon/finance/`. Top-level Finance nav button + status pills (Planning/In Progress/Done) derived from frontmatter.

**Architecture:** Three sub-areas isolated under sub-folders within one `module_directory: "finance"`. Budgets + Paychecks = flat dated entities (mirror to-do/journal pattern). Status derivation centralized in `FinanceStatus.derive(page, type)` shared helper.

**Tech Stack:** JSON manifests, Templater-token templates, CustomJS classes (no compile step), Dataviewjs callsites via customjs-guard, Node-harness install via `platform/test/run-install.js`, Cmd+R Obsidian for renderer reload.

### Task T2.3: Write `helpers/finance-status.js` (FinanceStatus shared helper)

**Files:** Create: `platform/blueprints/finance/helpers/finance-status.js`

```js
class FinanceStatus {
    /**
     * Derive status pill {label, tone} for a finance entity.
     * @param {Object} page - dataview page (frontmatter via page.* fields)
     * @param {"budget"|"paycheck"|"invoice"} type
     * @returns {{label: string, tone: "muted"|"warn"|"success"}}
     */
    derive(page, type) {
        const today = window.moment().startOf("day");
        if (type === "budget") {
            const month = page.budget_month;
            if (!month) return { label: "Planning", tone: "muted" };
            const m = window.moment(month, "YYYY-MM", true);
            if (!m.isValid()) return { label: "Planning", tone: "muted" };
            if (m.isBefore(today, "month")) return { label: "Done", tone: "success" };
            if (m.isSame(today, "month"))   return { label: "In Progress", tone: "warn" };
            return { label: "Planning", tone: "muted" };
        }
        if (type === "paycheck") {
            const start = page.pay_period_start ? window.moment(String(page.pay_period_start), "YYYY-MM-DD", true) : null;
            const expenses = Array.isArray(page.expenses) ? page.expenses : [];
            const allPaid = expenses.length > 0 && expenses.every(e => e.paid === true);
            if (allPaid) return { label: "Done", tone: "success" };
            if (start && start.isValid() && !today.isBefore(start, "day")) return { label: "In Progress", tone: "warn" };
            return { label: "Planning", tone: "muted" };
        }
        if (type === "invoice") {
            if (page.submitted_date) return { label: "Done", tone: "success" };
            const hours = Number(page.hours || 0);
            if (hours > 0) return { label: "In Progress", tone: "warn" };
            return { label: "Planning", tone: "muted" };
        }
        return { label: "Planning", tone: "muted" };
    }
}
```

```js
class BudgetsCards {
    async render(dv, opts) {
        opts = opts || {};
        const filter = opts.filter || "all";  // "all" | "pending"
        const pages = dv.pages('"beacon/finance/budgets"')
            .where(p => p.type === "budget")
            .array()
            .sort((a, b) => String(b.budget_month || "").localeCompare(String(a.budget_month || "")));
        const filtered = filter === "pending"
            ? pages.filter(p => customJS.FinanceStatus.derive(p, "budget").label !== "Done")
            : pages;
        await customJS.BeaconCards.render(dv, {
            pages: filtered,
            layout: "row",
            title: p => `Budget — ${p.budget_month || p.file.name}`,
            subtitle: p => {
                const cats = Array.isArray(p.categories) ? p.categories : [];
                const planned = cats.reduce((s, c) => s + Number(c.planned || 0), 0);
                const actual  = cats.reduce((s, c) => s + Number(c.actual  || 0), 0);
                return { text: `${cats.length} categories`, secondaryText: `Planned $${planned.toLocaleString()} · Actual $${actual.toLocaleString()}` };
            },
            badges: p => [customJS.FinanceStatus.derive(p, "budget")],
            targetFn: p => p.file.path,
        });
    }
}
```

### Task T2.11: Write `helpers/invoice-nav-buttons.js` (per-invoice context nav)

**Pattern:** Mirror `platform/blueprints/trips/helpers/trip-nav-buttons.js` simplified — auto-detect via path prefix `beacon/finance/invoices/<YYYY-MM>/`. Render buttons for: Invoice / Time-Log / Board / Finance Hub. Hide the button matching the active file.

**Files:** Create: `platform/blueprints/finance/helpers/invoice-nav-buttons.js`

```js
class InvoiceNavButtons {
    async render(dv) {
        const path = dv.current()?.file?.path || "";
        const m = path.match(/^beacon\/finance\/invoices\/(\d{4}-\d{2})\//);
        if (!m) return;  // not in an invoice ecosystem
        const month = m[1];
        const root = `beacon/finance/invoices/${month}`;
        const targets = [
            { label: "Invoice",      target: `${root}/Invoice-${month}.md` },
            { label: "Time Log",     target: `${root}/Time-Log-${month}.md` },
            { label: "Board",        target: `${root}/board/Board-${month}.md` },
            { label: "Finance Hub",  target: `beacon/finance/Finance.md` },
        ].filter(t => t.target !== path);
        const row = dv.container.createEl("div");
        row.style.cssText = "display: flex; gap: 8px; flex-wrap: wrap; margin: 8px 0;";
        for (const t of targets) {
            const btn = row.createEl("button");
            btn.textContent = t.label;
            btn.style.cssText = "cursor: pointer; padding: 6px 12px; border-radius: 6px; font-size: 0.85em;";
            btn.onclick = () => app.workspace.openLinkText(t.target, "");
        }
    }
}
```
