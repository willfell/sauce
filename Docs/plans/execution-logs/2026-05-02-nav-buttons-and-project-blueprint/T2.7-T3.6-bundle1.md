> - `boards/planning/...` — path shape assumptions throughout `project-nav-buttons.js` and the dataview source query in `planning-board-projects.js` (`dv.pages('"boards/planning"')`).
>
> **Templating-style strings (NOT installer vars):**
> - `project-nav-buttons.js` lines 177–209 use `{{DATE}}`, `{{TASK_PARENT_PATH}}`, `{{ALIAS}}`, `{{DATE_TAG}}`, `{{TASK_NAME}}`, `{{NEW_NOTE_FOLDER}}` as runtime `replaceAll` placeholders for template-string substitution. These are intentional and must NOT be confused with installer-time `{{rules_path}}` / `{{scripts_path}}` markers.
