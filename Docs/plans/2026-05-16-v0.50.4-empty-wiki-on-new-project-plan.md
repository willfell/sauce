### S1 — entity-create mechanism: ensure embedded-subfolder parent dir

**File:** `platform/mechanisms/entity-create/entity-create.js` (`_createExtra` near line 591)
**Change:** after computing `xPath`, ensure the parent directory of `xPath` if it differs from `xFolder` (filename_pattern with embedded slash case).
**Manifest bump:** `platform/mechanisms/entity-create/manifest.json` `0.3.1` → `0.3.2`.

Diff sketch:
```js
const xPath = `${xFolder}/${filename}`;
if (app.vault.getAbstractFileByPath(xPath)) return;
// Guard: filename_pattern may itself embed a subfolder (e.g. "wiki/Wiki.md").
const lastSlash = xPath.lastIndexOf("/");
const xParent = lastSlash > 0 ? xPath.substring(0, lastSlash) : "";
if (xParent && xParent !== xFolder) await this._ensureFolder(xParent);
```

**Commit:** `fix(entity-create): v0.3.2 — ensure parent dir when filename_pattern embeds subfolder`
