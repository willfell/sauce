# v0.39.0 — Projects Org-Chart Foundation Implementation Plan

> **For the implementing agent:** REQUIRED SUB-SKILL: Use `superpowers:executing-plans` (inline) or `superpowers:subagent-driven-development` (fresh subagent per task) to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `products@0.1.0` + `teams@0.1.0` blueprints, bump `project@1.5.0 → 1.6.0` with status / status_changed_at / teams[] / products[] frontmatter + Tier 1 action buttons + filter/group/sort hub upgrades, add `sauce migrate-projects` CLI verb, dogfood via workshop self-install + α-seeds for products + teams.

**Architecture:** Three blueprints, each owning its own `spice/<dir>/` module. `products` is the apex (no upward link). `teams` declares a required `product:` wikilink. `project` gains optional `teams[]` + `products[]` lists. Hub rendering reuses the existing `BeaconCards` (cards mechanism) for layout; new customjs classes per blueprint (`ProductsHubCards`, `ProductPageCards`, `ProductActionButtons`, `TeamsHubCards`, `TeamPageCards`, `TeamActionButtons`, upgraded `ProjectsHubCards`, NEW `ProjectActionButtons`). Buttons use the existing `accent-button` mechanism for high-emphasis action rows + the `@inquirer/prompts` shape Templater already exposes (suggester / multi-suggester via `tp.system.suggester`). Status enum is enforced via the existing validator's `matches` regex predicate — no validator bump needed.

**Tech Stack:** Node 18+, Obsidian Dataview, Obsidian Templater, sauce's existing mechanisms (`cards@0.2.4`, `nav-buttons@2.6.0`, `accent-button@0.1.0`, `customjs-guard@1.0.0`, `validator@0.1.2`). NEW infra from v0.38.0 we lean on: `sauce seed` CLI + per-blueprint `seed/` dirs, `run-integration-smoke.js`, `release:preflight` npm script, `check-version-sync.js` gate.

**Design doc:** `Docs/plans/2026-05-12-projects-org-and-lifecycle-design.md` (commit `1232a29`). Read it first.

### Task S1.3: Write ProductsHubCards customjs class

**Files:**
- Create: `platform/blueprints/products/scripts/products-hub-cards.js`

Mirrors `PeopleHubCards` shape. Sort by recently-touched-project descending (computed from inbound wikilinks); fall back to alphabetical when there are no project links.

- [ ] **Step 1: Write the customjs class**

```javascript
/**
 * Products Hub Cards (CustomJS)
 * Renders the hub at spice/products/Products.md. One card per product note.
 * Cards show: name, description, team count, active-project count, last-touched recency.
 * Sort: recent activity first (most-recently-touched project), alphabetical tiebreaker.
 *
 * Usage in DataviewJS:
 *   await dv.view("ranch/views/customjs-guard", { class: "ProductsHubCards" });
 */
class ProductsHubCards {
  async render(dv, opts) {
    if (dv.container.closest(".markdown-embed")) return;

    const products = dv.pages('"spice/products"')
      .where(p => p.file.name !== "Products" && p.type === "product");

    if (!products.length) {
      const empty = dv.container.createEl("div", {
        text: "No products yet — use /products or click + New Product to create your first."
      });
      empty.style.cssText = "color: var(--text-muted); font-style: italic; padding: 8px;";
      return;
    }

    // Pre-compute team + project counts + last-touched per product
    const teams = dv.pages('"spice/teams"').where(t => t.type === "team");
    const projects = dv.pages('"spice/projects"').where(p => p.type === "project");
    const ACTIVE = new Set(["idea", "planning", "in-progress", "blocked"]);

    const enriched = products.map(p => {
      const productLink = p.file.link.path;
      const memberTeams = teams.where(t => t.product && t.product.path === productLink);
      const memberTeamLinks = new Set(memberTeams.map(t => t.file.link.path));
      const memberProjects = projects.where(pr =>
        (pr.products || []).some(link => link.path === productLink) ||
        (pr.teams || []).some(link => memberTeamLinks.has(link.path))
      );
      const activeProjects = memberProjects.where(pr => ACTIVE.has(pr.status));
      // last-touched: max status_changed_at across member projects
      const dates = memberProjects.map(pr => pr.status_changed_at || pr.created || "1970-01-01");
      const lastTouched = dates.array().sort().pop() || null;
      return { product: p, teamCount: memberTeams.length, activeCount: activeProjects.length, lastTouched };
    });

    // Sort: lastTouched desc, alphabetical tiebreaker
    enriched.sort((a, b) => {
      const at = a.lastTouched || "1970-01-01";
      const bt = b.lastTouched || "1970-01-01";
      if (at !== bt) return bt.localeCompare(at);
      return a.product.file.name.localeCompare(b.product.file.name);
    });

    await customJS.BeaconCards.render(dv, {
      pages: enriched.map(e => e.product),
      layout: "row",
      columns: 1,
      title: (p) => p.file.name,
      subtitle: (p) => p.description || null,
      meta: (p) => {
        const e = enriched.find(x => x.product.file.path === p.file.path);
        const teamPart = `${e.teamCount} team${e.teamCount === 1 ? "" : "s"}`;
        const projPart = `${e.activeCount} active project${e.activeCount === 1 ? "" : "s"}`;
        const datePart = e.lastTouched ? `· last touched ${e.lastTouched}` : "";
        return `${teamPart} · ${projPart} ${datePart}`.trim();
      }
    });
  }
}
```

```javascript
/**
 * Product Action Buttons (CustomJS)
 * Renders on the Products hub (Products.md) the + New Product button.
 * Uses the accent-button mechanism for styling.
 *
 * Usage:
 *   await dv.view("ranch/views/customjs-guard", { class: "ProductActionButtons" });
 */
class ProductActionButtons {
  async render(dv, opts) {
    if (dv.container.closest(".markdown-embed")) return;
    if (!customJS.BeaconAccentButton) {
      const warn = dv.container.createEl("div", { text: "accent-button mechanism not loaded" });
      warn.style.color = "var(--text-error)";
      return;
    }

    await customJS.BeaconAccentButton.render(dv, {
      label: "+ New Product",
      onClick: async () => {
        // Invoke Templater's "Create new note from template" with our Product template
        const tpFolder = "ranch/templates";
        const tpFile = `${tpFolder}/Template, Product.md`;
        const target = app.vault.getAbstractFileByPath(tpFile);
        if (!target) {
          new Notice("Template, Product.md not found at " + tpFile);
          return;
        }
        // Use Templater public API
        const templater = app.plugins.plugins["templater-obsidian"];
        if (!templater) {
          new Notice("Templater plugin not enabled");
          return;
        }
        await templater.templater.create_new_note_from_template(
          target,
          app.vault.getAbstractFileByPath("spice/products"),
          undefined,  // filename — Templater prompt fills via tp.file.rename inside template
          true
        );
      }
    });
  }
}
```

### Task S3.3: Write TeamsHubCards customjs class (groups by product, status histogram chip)

**Files:**
- Create: `platform/blueprints/teams/scripts/teams-hub-cards.js`

- [ ] **Step 1: Write the customjs class**

```javascript
/**
 * Teams Hub Cards (CustomJS)
 * Renders the hub at spice/teams/Teams.md. Cards grouped by Product (section per product).
 * Each card: team name, description, active-project count, status histogram chip
 * (e.g. "3 in-progress · 2 planning · 1 blocked · 2 done").
 * Filter chip stub: by product (multi-select) — Phase 1 ships group-by-only; chip
 * UI lands when ProjectsHubCards's chip pattern is settled in S6 (then refactor).
 *
 * Usage:
 *   await dv.view("ranch/views/customjs-guard", { class: "TeamsHubCards" });
 */
class TeamsHubCards {
  async render(dv, opts) {
    if (dv.container.closest(".markdown-embed")) return;

    const teams = dv.pages('"spice/teams"').where(t => t.file.name !== "Teams" && t.type === "team");
    if (!teams.length) {
      const empty = dv.container.createEl("div", {
        text: "No teams yet — use /teams or click + New Team. Each team needs a product: wikilink."
      });
      empty.style.cssText = "color: var(--text-muted); font-style: italic; padding: 8px;";
      return;
    }

    const projects = dv.pages('"spice/projects"').where(p => p.type === "project");
    const STATUS_ORDER = ["in-progress", "planning", "blocked", "idea", "done", "superseded", "cancelled"];
    const ACTIVE = new Set(["idea", "planning", "in-progress", "blocked"]);

    // Compute per-team: active count + status histogram
    const enriched = teams.map(t => {
      const teamLinkPath = t.file.link.path;
      const memberProjects = projects.where(p => (p.teams || []).some(link => link.path === teamLinkPath));
      const hist = {};
      memberProjects.forEach(p => { hist[p.status] = (hist[p.status] || 0) + 1; });
      const histStr = STATUS_ORDER
        .filter(s => hist[s])
        .map(s => `${hist[s]} ${s}`)
        .join(" · ");
      const activeCount = memberProjects.where(p => ACTIVE.has(p.status)).length;
      return { team: t, activeCount, histStr };
    });

    // Group by product link path
    const groups = new Map();
    enriched.forEach(e => {
      const key = e.team.product ? e.team.product.path : "(no product)";
      if (!groups.has(key)) groups.set(key, []);
      groups.get(key).push(e);
    });

    // Sort group keys by product name (alphabetical)
    const sortedKeys = [...groups.keys()].sort();

    for (const key of sortedKeys) {
      const header = dv.container.createEl("h2", { text: key === "(no product)" ? key : `[[${key.replace(/\.md$/, "")}]]` });
      header.style.cssText = "margin-top: 16px; margin-bottom: 4px;";
      const groupTeams = groups.get(key);
      groupTeams.sort((a, b) => a.team.file.name.localeCompare(b.team.file.name));

      await customJS.BeaconCards.render(dv, {
        pages: groupTeams.map(e => e.team),
        layout: "row",
        columns: 1,
        title: (t) => t.file.name,
        subtitle: (t) => t.description || null,
        meta: (t) => {
          const e = enriched.find(x => x.team.file.path === t.file.path);
          const active = `${e.activeCount} active`;
          return e.histStr ? `${active} · ${e.histStr}` : active;
        }
      });
    }
  }
}
```

 *   await dv.view("ranch/views/customjs-guard", { class: "ProjectActionButtons" });

class ProjectActionButtons {
  async render(dv, opts) {
    if (dv.container.closest(".markdown-embed")) return;
    const page = dv.current();
    if (!page || page.type !== "project") return;

    const row = dv.container.createEl("div");

    btn.style.cssText = "padding:4px 10px;background:var(--interactive-accent);color:var(--text-on-accent);border:none;border-radius:4px;cursor:pointer;";
    btn.addEventListener("click", async () => {
      const STATUSES = ["idea", "planning", "in-progress", "blocked", "superseded", "cancelled", "done"];
      const tp = app.plugins.plugins["templater-obsidian"]?.templater?.current_functions_object ||
                 (await app.plugins.plugins["templater-obsidian"]?.templater?.create_running_config_for_active(null));
      if (!tp) { new Notice("Templater not available"); return; }
      const choice = await tp.system.suggester(STATUSES, STATUSES, false, `New status (current: ${page.status})`);
      if (!choice) return;
      const today = window.moment().format("YYYY-MM-DD");
      await this._patchFrontmatter(page.file.path, { status: choice, status_changed_at: today });
      new Notice(`Status: ${choice} (was ${page.status})`);
    });

  async _patchFrontmatter(filePath, patch) {
    const file = app.vault.getAbstractFileByPath(filePath);
    if (!file) { new Notice("File not found: " + filePath); return; }
    const content = await app.vault.read(file);
    const fmMatch = content.match(/^---\n([\s\S]*?)\n---/);
    if (!fmMatch) { new Notice("No frontmatter found"); return; }
    let fmBody = fmMatch[1];
    for (const [key, value] of Object.entries(patch)) {
      const re = new RegExp(`^${key}:\\s*.*$`, "m");
      const newLine = Array.isArray(value)
        ? `${key}:\n${value.map(v => `  - ${v}`).join("\n")}`
        : `${key}: ${typeof value === "string" ? (value.includes('"') ? value : value) : value}`;
      if (re.test(fmBody)) {
        fmBody = fmBody.replace(re, newLine.split("\n")[0]); // single-line replace for scalar
        if (Array.isArray(value)) {
          // crude list rewrite: replace key: + following indented lines
          const listRe = new RegExp(`^${key}:[\\s\\S]*?(?=\\n[a-z_]+:|$)`, "m");
          fmBody = fmBody.replace(listRe, newLine);
        }
      } else {
        fmBody += `\n${newLine}`;
      }
    }
    const newContent = content.replace(/^---\n[\s\S]*?\n---/, `---\n${fmBody}\n---`);
    await app.vault.modify(file, newContent);
  }

// platform/cli/cmd-migrate-projects.js
// CLI verb: `sauce migrate-projects [--vault <path>] [--dry-run]`
// Backfills status: in-progress and status_changed_at: <created> on every
// type:project note that lacks them. Idempotent.

const fs = require("fs");
const path = require("path");

function parseFrontmatter(content) {
  const m = content.match(/^---\n([\s\S]*?)\n---\n/);
  if (!m) return { fm: null, body: content, raw: null };
  const fm = {};
  m[1].split("\n").forEach(line => {
    const kv = line.match(/^([a-z_]+):\s*(.*)$/);
    if (kv) fm[kv[1]] = kv[2].replace(/^"|"$/g, "");
  });
  return { fm, body: content.slice(m[0].length), raw: m[1] };
}

function injectFrontmatter(content, patch) {
  const m = content.match(/^---\n([\s\S]*?)\n---\n/);
  if (!m) {
    // No frontmatter — create one
    const lines = Object.entries(patch).map(([k, v]) => `${k}: ${v}`).join("\n");
    return `---\n${lines}\n---\n\n${content}`;
  }
  let fmBody = m[1];
  for (const [key, value] of Object.entries(patch)) {
    const re = new RegExp(`^${key}:\\s*.*$`, "m");
    if (re.test(fmBody)) continue;  // already present — leave
    fmBody += `\n${key}: ${value}`;
  }
  return content.replace(m[0], `---\n${fmBody}\n---\n`);
}

function walk(dir, fn) {
  if (!fs.existsSync(dir)) return;
  for (const entry of fs.readdirSync(dir, { withFileTypes: true })) {
    const full = path.join(dir, entry.name);
    if (entry.isDirectory()) walk(full, fn);
    else if (entry.isFile() && entry.name.endsWith(".md")) fn(full);
  }
}

async function run(opts = {}) {
  const vault = path.resolve(opts.vault || process.cwd());
  const dryRun = !!opts.dryRun;
  const today = new Date().toISOString().slice(0, 10);

  const projectsDir = path.join(vault, "spice/projects");
  if (!fs.existsSync(projectsDir)) {
    console.log(`No spice/projects/ in ${vault} — nothing to migrate.`);
    return { migrated: 0, alreadyOk: 0 };
  }

  let migrated = 0;
  let alreadyOk = 0;
  const targets = [];
  walk(projectsDir, file => {
    const content = fs.readFileSync(file, "utf8");
    const { fm } = parseFrontmatter(content);
    if (!fm || fm.type !== "project") return;
    const needs = {};
    if (!fm.status) needs.status = "in-progress";
    if (!fm.status_changed_at) needs.status_changed_at = fm.created || today;
    if (!fm.teams) needs.teams = "[]";
    if (!fm.products) needs.products = "[]";
    if (Object.keys(needs).length === 0) {
      alreadyOk++;
      return;
    }
    targets.push({ file, needs });
  });

  for (const { file, needs } of targets) {
    if (dryRun) {
      console.log(`[dry-run] ${file}: would patch ${Object.keys(needs).join(", ")}`);
    } else {
      const content = fs.readFileSync(file, "utf8");
      const patched = injectFrontmatter(content, needs);
      fs.writeFileSync(file, patched);
      console.log(`migrated: ${file} (added ${Object.keys(needs).join(", ")})`);
    }
    migrated++;
  }

  console.log(`\n${migrated} project${migrated === 1 ? "" : "s"} ${dryRun ? "would be" : ""}migrated, ${alreadyOk} already had status.`);
  return { migrated, alreadyOk };
}

module.exports = { run };
