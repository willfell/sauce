- `.sauce-group-row-title` gets `width: 100%; white-space: normal; overflow: visible; text-overflow: clip` — title now wraps onto multiple lines instead of ellipsizing.
- `.sauce-group-row-meta` gets `width: 100%; justify-content: flex-start` — meta wraps below the title with left-anchored tokens.
- `.sauce-group-row-meta > time` overrides the desktop anchor: `min-width: auto; text-align: left` so the time stays inline with the rest of the meta tokens on mobile.

Fixes the v0.70.0 regression where wide-meta rows (e.g., kanban "To Do Board" with a breadcrumb) collapsed the title to 0 px width on narrow viewports — the row appeared title-less.

## Lessons

- **Workshop catalogue (`platform/manifest.json`) and per-component manifest (`platform/blueprints/<name>/manifest.json`) must bump together.** The installer reads the catalogue's `blueprints[]` / `mechanisms[]` arrays to learn workshop-side versions; bumping only the per-component manifest leaves the catalogue stale and the installer skips the item with `subscription pins X@0.10.1 but workshop has 0.10.0`. Encountered during the v0.70.1 self-install — diagnosed via `--verbose`. Worth capturing in `build-test-verify.md`.
- **Flex `min-width: 0` on a title plus flex-grow doesn't save the title from disappearing when sibling meta is wider than the row.** The browser shrinks the title to 0 px and the `overflow: hidden; text-overflow: ellipsis;` chain shows nothing because there's no text-space to display anything in. On narrow viewports, switching the parent to `flex-direction: column` (stacking) is the correct fix; ellipsis truncation is a layout for tables, not for variable-width content.
- **The CSS `<time>` element with `font-variant-numeric: tabular-nums` is the right primitive for time alignment in lists.** Tabular nums make digits share fixed advance widths so "1:30 PM" and "12:22 PM" hit the same right edge. Combined with `min-width` + `text-align: right` on the `<time>` itself, time columns align vertically across rows without grid layout or DOM restructure.
