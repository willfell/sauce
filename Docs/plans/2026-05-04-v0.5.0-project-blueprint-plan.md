# v0.5.0 Project Blueprint Implementation Plan

> **For the implementing agent:** REQUIRED SUB-SKILL: Use `de:executing-plans` to implement this plan task-by-task. Pacing per post-v0.4.2 handoff: per-task subagent dispatch on installer-extending + blueprint-authoring + writeup tasks (the meaty ones); master-driven mechanical tasks. Stage-close `AskUserQuestion` checkpoints before each commit + push. TDD on any harness extensions (likely none this cycle). Cmd+R Obsidian after Node-harness install at S2 (3 CustomJS classes ship — gotcha 2). Pre-flight diff barebones `installed.json` vs `subscription.json` before S2 install (gotcha 9).

**Goal:** Replace `project@0.3.0` (`projects-legacy` stopgap) with `project@1.0.0` materialized under `beacon/projects/` per the module-directory invariant. Ports the accuris projects pattern: per-project atlas + structure + Kanban board, hierarchical tasks (task hub + notes + task-board + folder-style sub-task cards), workstream-tagged cards, and an All-Projects hub.

**Architecture:** Existing blueprint's helper classes are byte-identical copies of accuris source (633 + 249 + 184 LOC) — v0.5.0 is a **path-rewrite refactor**, not a fresh port. Surgical sed/edit pattern (`boards/planning/` → `beacon/projects/`; `Extras/Templates/` → `{{templates_path}}/`; `Extras/Scripts/customjs-guard` → `Docs/Meta/Views/customjs-guard`) plus one new `projects-hub` context in `ProjectNavButtons` for the New Project action. Manifest refactor drops the slash-command surface (per user pref — "Only nav-button on Projects hub note") + `module_directory` rename + `pre_install[delete]` for legacy file. No new installer mechanics.

**Tech Stack:** Node.js install harness (`platform/test/run-install.js`), Obsidian/Templater/CustomJS/Dataview runtime, JSON manifests, `applyTemplaterFolderTemplates` (v0.4.0) for Kanban-card folder-templates inherited at runtime per project (NOT static module_directory paths), `pre_install: [{type:"delete"}]` (v0.2.0) for legacy file retirement.

```javascript
async _createProject({ name, slug }) {
    const projectDir = `beacon/projects/${slug}`;
    if (!app.vault.getAbstractFileByPath(projectDir)) {
        await app.vault.createFolder(projectDir);
    }

    const tplBase = "Docs/Meta/Templater/Templates";
    const now = new Date();
    const pad = (n) => String(n).padStart(2, "0");
    const dateStr = `${now.getFullYear()}-${pad(now.getMonth() + 1)}-${pad(now.getDate())} ${pad(now.getHours())}:${pad(now.getMinutes())}`;
    const dateTag = `${now.getFullYear()}/${pad(now.getMonth() + 1)}/${pad(now.getDate())}`;

    const subs = (s) => s
        .replaceAll("{{NAME}}", name)
        .replaceAll("{{SLUG}}", slug)
        .replaceAll("{{DATE}}", dateStr)
        .replaceAll("{{DATE_TAG}}", dateTag);

    const writeTpl = async (tplName, destBasename) => {
        const tplFile = app.vault.getAbstractFileByPath(`${tplBase}/${tplName}`);
        if (!tplFile) {
            new Notice(`Template missing: ${tplBase}/${tplName}`);
            return null;
        }
        const tpl = await app.vault.read(tplFile);
        const content = subs(tpl);
        const targetPath = `${projectDir}/${destBasename}`;
        if (app.vault.getAbstractFileByPath(targetPath)) return targetPath;
        await app.vault.create(targetPath, content);
        return targetPath;
    };

    const atlasPath = await writeTpl("Template, Project.md", `${name}.md`);
    await writeTpl("Template, Project Structure.md", `${name} - Structure.md`);
    await writeTpl("Template, Project Board.md", `${slug}-board.md`);

    return atlasPath;
}
```
