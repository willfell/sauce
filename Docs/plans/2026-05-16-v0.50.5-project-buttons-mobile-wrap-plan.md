# v0.50.5 Project Nav-Buttons Mobile Wrap Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make `ProjectNavButtons` button row wrap to multiple lines when buttons don't fit on a single row (fixes mobile clutter).

**Architecture:** Pure CSS change in one render method. `flex-wrap: nowrap` → `wrap`; `flex: 1` → `flex: 0 1 auto`; add `white-space: nowrap` to label spans so the button wraps to a new flex row instead of the label wrapping inside.

**Tech Stack:** Obsidian Dataview helper (CustomJS class), vanilla DOM + CSS, Node test harness.

**Design doc:** `Docs/plans/2026-05-16-v0.50.5-project-buttons-mobile-wrap-design.md`

**Cadence:** Controller-direct edits (no sub-agents — single-file PATCH).
