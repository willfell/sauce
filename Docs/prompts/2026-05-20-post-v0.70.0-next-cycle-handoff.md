## Lessons worth highlighting

- **Buried test-pin grep before preflight.** v0.70.0 hit three buried pins across two harnesses (AF-V065, FA6-MANIFEST-activity-feed, FA6-MANIFEST-daily, FA4-MANIFEST-daily). Future cycles bumping a manifest version should run `grep -rn '"<old-version>"' platform/test/` first.
- **Renderer bypassing BeaconCards is sometimes the right call.** The framed surface in `_renderFramedGroup` ignores cards entirely for inner rows. When a layout doesn't match the cards primitive, render direct and pass the same `getTitle`/`metaBuilder` contract.
